# AfricaZero 产品路线图：下一阶段设计

> 核心使命：不是"关税计算器"，而是**"告诉我能不能做、找谁做、怎么做"**的非洲进口决策平台。
>
> 2026年5月1日，中国对53个非洲国家100%税目零关税政策正式生效。AfricaZero站在历史窗口期，需要从**工具**进化为**平台**。

---

## 一、方向一：AI 原产地证书引导办理

### 1.1 问题本质

用户在原产地自测（/origin-check）拿到 AI 判定结果后，最大的困惑是：

> *"AI说我可能符合条件，但我要怎么把一张纸变成海关认可的证书？"*

**现状**：用户只能看到 AI 给出的"可能符合/不符合"，但不知道：
- 需要准备哪些文件
- 去哪里办（贸促会/海关）
- 流程是什么、要跑几趟
- 整个流程要多少钱、要多久

**核心差距**：AfricaZero 回答了"能不能做"，但用户真正需要的是"**怎么做**"。

#### AI 证书文件生成器（核心创新）

用户在完成原产地自测后，系统根据 AI 分析结果，**自动生成**：

**A. 原产地证明申请书（Exporter's Declaration）**
- 根据用户填写的货物信息、HS编码、原产国自动填充
- 可导出为 PDF / Word，方便打印盖章

**B. 加工工序说明文件（Processing Description）**
- 基于用户在 origin-check 页面填写的加工步骤
- AI 润色为正式法律语言
- 标注适用规则（CTC / VA / SP）

**C. 所需文件清单 + 时间线**
```
贸促会办理流程（约5-10个工作日，总费用 ¥300-600）：
Day 1-2: 非洲出口商在当地商会申请原产地证（CO）
Day 3-7: 文件快递至中国（约5-7天）
Day 8-10: 中国进口商向贸促会提交认证申请
Day 10:   领取认证证书，进口清关使用
```

#### Step-by-Step 引导流程

```
Step 1: 确认资格（继承 origin-check 结果）
         ├─ AI 判定结果：符合/不符合
         ├─ 适用规则：CTC / VA(40%) / SP
         └─ 置信度：85%

Step 2: 收集文件（按清单勾选）
         ├─ 自动生成的文件（AfricaZero AI）
         ├─ 需要第三方出具的文件（链接到说明页）
         └─ 进度追踪：已准备 X/Y 项

Step 3: 选择办理机构
         ├─ 贸促会（推荐，速度快，¥300-600）
         ├─ 海关（成本低，但周期长）
         └─ 第三方代理（费用高，适合没有贸促会分支的城市）

Step 4: 在线提交/预约
         ├─ 贸促会在线系统预审（API接入）
         └─ 生成预审码，线下快速办理

Step 5: 跟踪进度 + 获取证书
         ├─ 证书编号查询
         └─ 证书模板预览 + 下载
```

#### 贸促会 API 集成（Phase 2）

贸促会提供企业端 API（ccpit.org），未来可实现：
- 在线提交电子文件预审
- 实时查询办理进度
- 数字签名 + 电子证书（无需快递纸质文件）

## 二、方向二：供应商发现与验证体系

### 2.1 问题本质

> *"关税算清楚了，但我去哪找靠谱的埃塞俄比亚咖啡供应商？"*

**现状**：
- 选品页（/products）解决了"选什么"的问题
- 但用户看完品类后，只能去 Google 找供应商，信息极度不透明
- 非洲供应商验证成本极高：沟通成本、语言障碍、文化差异、付款风险

**市场空白**：没有一个平台同时具备：
1. 关税/原产地合规知识（知道哪些供应商能帮买家拿到零关税）
2. 供应商信用评估
3. 可信的交易撮合

---

### 2.2 产品设计：非洲供应商数据库

#### 数据来源（冷启动策略）

供应商数据是最难的部分，采用**三步走策略**：

**Phase 1：手工数据 + AI 扩充（0→100家）**
- 从贸促会、埃塞俄比亚驻华使馆商务处等公开渠道搜集出口商名单
- 从阿里巴巴国际站/Amanbo 公开商家数据中标注非洲供应商
- AI 辅助清洗和补全（公司名、地址、联系方式、主要产品、出口经验）

**Phase 2：用户贡献（100→1000家）**
- 付费用户可提交自己合作的供应商
- 提交需审核：营业执照 + 出口中国历史记录
- 贡献者获得平台积分（可抵扣订阅费）

**Phase 3：官方数据接入（1000+家）**
- 接入非洲各国商会 API
- 贸促会双边认证企业数据库
- 区块链溯源：供应商货物溯源链上链

│    └─ 零关税原产地证书办理成功率                     │
│                                                     │
│ 3. 用户评价（20%）                                   │
│    ├─ 货物质量评分                                  │
│    ├─ 按时交货率                                    │
│    └─ 沟通响应速度                                  │

**评级标签**：

- ✅ **已验证**：通过基础审核，有出口中国记录
- 🆕 **新加入**：审核通过但暂无交易记录（建议小额试单）
- ⚠️ **待观察**：有用户投诉，调查中

### 2.3 贸易撮合平台

在供应商数据库基础上，构建轻量级撮合功能：

**A. 询价单（RFQ）系统**

- 资金托管：买方付款至平台，买方确认收货后放款给卖方
- 纠纷仲裁：平台介入交易纠纷
- 接入支付宝/微信跨境支付

### 2.4 与竞品的差异化

| 维度 | Egatee/Amanbo | AfricaZero（目标） |
|------|---------------|-------------------|
| 核心能力 | 撮合为主 | 关税+合规+撮合 |
| 原产地证书 | 不支持 | 全程支持 |
| 零关税合规 | 不支持 | 核心优势 |
| 供应商到岸成本 | 不计算 | 一键计算 |
| 目标用户 | 出口商为主 | **进口商为主** |
| 数据深度 | 泛非 | 聚焦零关税品类 |

**核心洞察**：所有竞品都在帮非洲卖家把货卖给中国买家，但**没有人帮中国买家做进口决策**。AfricaZero 的供应商发现体系是**全球唯一以"中国进口商"为中心的非洲贸易撮合平台**。

## 三、补充功能：物流成本实时估算

### 3.1 为什么重要

物流成本占 CIF（到岸价）的 20-40%，且波动极大：
- 埃塞俄比亚→中国：无直达船，需经 Djibouti（吉布提）或 Durban（德班）
- 集装箱 20GP：约 $1,500-3,500（随淡季/旺季波动）
- 散货船按吨计：$0.05-0.15/kg

现有成本精算器用的是**固定估算值**，无法反映实时市场。

#### 数据来源
