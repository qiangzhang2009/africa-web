## 步骤3：自动校验数据（防止错税率、错HS）

```Python

# check_data.py
import sqlite3

conn = sqlite3.connect("hs_code_db.sqlite3")
c = conn.cursor()

# 检查HS是否10位、税率0~1之间
c.execute("SELECT hs_code, mfn_rate FROM hs_code WHERE LENGTH(hs_code)!=10 OR mfn_rate<0 OR mfn_rate>1")
errors = c.fetchall()

if errors:
    print("❌ 数据错误：", errors)
else:
    print("✅ 数据校验通过")
conn.close()
```
