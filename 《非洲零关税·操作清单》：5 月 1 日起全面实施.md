# 二、原产地规则（官方细则，直接背）

## 1. 完全原产（最简单，直接过）

符合以下任一，**直接认定非洲原产**：

- 非洲土地/海洋里挖的、采的：矿产品、农产品、渔获

- 完全用非洲本地原料生产、无任何外国成分

- 非洲出生、饲养的动物及其产品

**适用：咖啡、可可、腰果、芝麻、鲜花、水果、矿产、木材、橡胶、棉花**

## 2. 非完全原产（加工过的商品）

满足**二选一**即可：

1. **4位税号改变**

原料和成品前4位HS编码不一样

1. **外来原料≤60%**

非非洲原料价值 ≤ 产品FOB总价的 **60%**，且**最后工序在非洲完成**

# 三、原产地证书：怎么拿、谁来办、长什么样

## 1. 证书类型

- **非洲53国特惠原产地证书**（专用）

- 或 **FORM A**（部分国家仍可用）

- **电子证书**同等有效

## 2. 谁办、在哪办

- **由非洲出口商在当地办理**

- 签发机构：

    - 该国**海关**

    - **商务部/经济部**

    - **工商会/贸促会**（官方授权机构）

## 3. 办理材料（出口商准备）

- 商业发票

- 装箱单

- 提单/空运单

- 产品生产/加工说明

- 原料来源证明（如涉及非非洲原料）

- 原产地证书申请表

## 4. 证书必须包含内容

- 出口商信息

- 进口商信息（你）

- **HS编码**

- **原产国**

- FOB价值

- 原产地标准（如“WO”完全原产）

- 官方签章、签发日期

## 5. 有效期

**自签发之日起 1 年**

# 四、国内报关申报零关税 完整步骤（照着填）

## 1. 报关时必填项

- **贸易方式**：一般贸易 / 跨境电商 / 保税备货

- **优惠政策**：选择 **非洲53国零关税特惠**

- **税率类型**：特惠税率

- **税率**：**0%**

- **原产地证书编号**：必须填写

## 2. 必须上传的单证

1. **原产地证书**（核心）

2. 商业发票

3. 装箱单

4. 提单/空运单

5. 检验检疫证书（食品、生鲜、动植物）

6. 其他许可（危化品、特殊商品）

## 3. 通关绿色通道（官方便利）

- 非洲输华货物**优先查验、优先放行**

- 单证齐全：**24小时内办结**

- 生鲜/农食：**即检即放**，通关时间压缩到**120小时内**

- 货值≤**6000元**：可**免原产地证书**

# 五、最关键：合同必须加这一条（防坑）

你和非洲供应商签合同时，**强制加这一句**：

“卖方保证货物为非洲53国原产，并负责提供中国海关认可的官方原产地证书，确保买方享受零关税待遇。如因原产地问题无法享受零关税，卖方承担全部关税及损失。”

# 六、常见违规红线（千万别碰）

- 不能用**第三国货物冒充非洲货**

- 不能**买假证、PS证书**

- 不能**洗产地、中转改标签**

- 不能**虚假申报原产国**

后果：

- 追缴税款 + 罚款

- 列入失信名单

- 3年内禁止享受优惠政策

# 七、极简执行流程（你只需要按顺序做）

1. 选品 → 确认在**中国准入**

2. 找非洲供应商 → 合同写明**原产地责任**

6. 放行 → 提货销售
