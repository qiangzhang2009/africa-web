### 1.2 增值税率（VAT）

**法律依据**：《中华人民共和国增值税法》（2026年1月1日起施行）及海关总署配套规定。

税率按商品属性分两档：

| 类别 | 增值税率 | 适用商品 |
|------|---------|---------|
| **农产品**（初级农产品，进口环节） | **9%** | 生咖啡豆、可可豆、腰果、杏仁、榛子、芝麻等 |
| **加工品**（经加工制造） | **13%** | 已焙炒咖啡豆、可可膏/脂/粉、速溶咖啡、茶、矿砂 |

> **注**：农产品（09-14章等）适用9%优惠税率；经过加工制造的商品（18章可可制品、20-22章加工食品等）适用13%税率。

### 1.4 非洲内部 AfCFTA 关税政策

**数据来源**：AfCFTA Rules of Origin Manual（官方发布，`etariff.au-afcfta.org`）。

关税削减分三档：
- 非敏感商品（90%税目）：最不发达国家10年内、其余国家5年内清零
- 敏感商品（7%税目）：保留5年后开始削减
- 排除商品（3%税目）：可不参与自由化

**原产地规则**：区域价值含量（RVC）≥ 40% 为通用方法之一；也可通过关税分类变化或特定加工工序满足条件。

### 2.2 保险费率

**数据来源**：《国际贸易术语解释通则 2020》（Incoterms 2020）及国际货运保险惯例（ICC 条款）。

计算公式使用 ICC(C) 最低险别惯例：

```
保险费 = FOB货值 × 0.5%
```

> 注：0.5% 为国际贸易中的一般货物最低投保比率（ICC(C) 条款），适用于普通散货。实际保费因货物类别、运输条件而异，高价值货物（咖啡生豆等）通常采用 ICC(A) 全险，费率约 0.8-1.0%。

## 四、计算逻辑详解

### 4.1 CIF 价格计算（关税计算器）

```
步骤1：FOB USD → CIF USD
  FOB（用户输入）
  + 运费 = 用户指定 freight_override 或 (参考运费率 × 采购量 kg)
  + 保险 = FOB × 0.5%
  = CIF USD

步骤2：USD → CNY（到岸价）
  FOB_CNY = FOB USD × 汇率
  运费_CNY = 运费 USD × 汇率
  保险_CNY = 保险 USD × 汇率

步骤3：关税
  关税税率 = 零关税（MFA白名单内）或 MFN税率（非建交国/非优惠）
  关税金额 = FOB_CNY × 关税税率

步骤4：增值税（从价税）
  计税基础 = FOB_CNY + 运费_CNY + 保险_CNY + 关税金额
  增值税 = 计税基础 × 增值税率（9%农产品 / 13%加工品）

步骤5：到岸总成本
  总成本 = FOB_CNY + 运费_CNY + 保险_CNY + 关税金额 + 增值税

步骤6：节省金额（对比 MFN）
  节省 = FOB_CNY × MFN税率（假设无零关税优惠时）
```

### 4.2 目的市场判断逻辑

```
目的地 = CN（中国）:
  原产国 ∈ 53个建交国 → 关税 = 0，origin_qualified = true
  原产国 ∉ 53个建交国 → 关税 = MFN税率，origin_qualified = false

目的地 = EU（欧盟）:
  原产国 ∈ EU EPA 国家列表 → 关税 = 0（需验证增值≥40%），origin_qualified = true
  原产国 ∉ EPA 列表 → 关税 = MFN税率

目的地 = AFCFTA（非洲内部）:
  所有非洲产国 → 关税 = 0（AfCFTA区内），origin_qualified = true
```

### 4.3 成本精算器（完整链路）

```
FOB货值 = 采购量(kg) × FOB单价(USD/kg)
国际运费 = 运费率(USD/kg) × max(采购量, 10kg)
保险 = FOB货值 × 0.5%
清关杂费 = 2000 CNY + 采购量 × 2 CNY/kg
关税 = 按零关税政策（非洲建交国）或 MFN税率
增值税 = (FOB_CNY + 运费_CNY + 保险_CNY + 关税) × VAT税率
→ 进口总成本

烘焙损耗 = 采购量 × 15%（生豆→熟豆水分损耗+熟豆损耗）
烘焙出品 = 采购量 - 烘焙损耗
分装包数 = floor(烘焙出品 / 0.227kg)
分装成本 = 包数 × ¥0.5/包
→ 综合成本

建议零售价 = max(参考价×0.8, 每包成本×2.5)
回本包数 = ceil(综合成本 / 建议零售价)
```

## 七、使用建议与免责声明

1. **FOB 货值**：请以实际采购合同或供应商报价为准，不同品质级别（AA/AB/SC等）价格差异显著。
2. **运费**：咖啡豆运输通常采用 LCL 拼箱，60kg 小批量运费约 $150-300USD，建议使用货代实际报价。
3. **汇率**：本工具使用参考汇率，实际结汇汇率请咨询银行。
4. **原产地认证**：零关税待遇需在进口清关时提交有效原产地证书（Form A 或相应优惠证书），工具显示"符合条件"不等于已获证书。
5. **政策时效**：关税税率基于2025年关税税则，2026年5月1日起零关税政策全面覆盖53国；税率如有调整，以海关最新公告为准。
