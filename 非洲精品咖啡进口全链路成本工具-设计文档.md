## 核心功能详解

### 1. 关税计算器（`/calculator`）

#### 用户输入
- **品类预设**：按产品形态分组（生豆 / 烘焙豆 / 速溶 / 坚果），一键填入 HS 编码、原产国、参考采购量
- **HS 编码**：支持实时联想搜索（调用后端 `/api/v1/hs-codes/search`），选择后自动填入
- **原产国**：两字母 ISO 代码（由预设自动填入）
- **目的地市场**：中国（零关税）/ 欧盟（EPA）/ 非洲内部（AfCFTA）
- **采购量（kg）**：用于精确计算运费摊薄（60kg 默认）
- **FOB 货值（USD）**：FOB 单价 × 采购量
- **国际运费方式**：海运拼箱（LCL）/ 空运拼货 / 整柜海运（FCL）/ 手动输入总运费
- **汇率 USD→CNY**：默认 7.25，支持手动锁定

#### 后端计算逻辑
```
FOB 货值（USD）
  + 国际运费 = 产国运费率 × 采购量  （或用户手动指定）
  + 保险 = FOB × 0.5%
→ CIF（到岸价）
  + 关税 = 零关税（建交国产）/ MFN 税率（非建交国）
  + 增值税 = (FOB + 运费 + 保险 + 关税) × 13%
→ 到岸总成本（CNY）
节省金额 = FOB × MFN 税率（CNY）
```

### 2. HS 编码查询（`/hs-lookup`）

- 支持中文名称、HS 编码前缀搜索
- 返回匹配结果（10位码、名称中文、MFN 税率、品类）
- 计算器内嵌 HS 联想，无需跳页

---

### 3. 原产地自测（`/origin-check`）

- 用户输入：商品名称、HS 编码、原产国、加工工序列表、原料来源
- 后端调用 DeepSeek API（需配置 `DEEPSEEK_API_KEY`）
- 无 API Key 时切换规则估算模式（置信度 0.7）

---

### 4. 国家信息（`/countries`）

- 返回所有非洲国家基本信息（代码、中英文名、是否参与 AfCFTA、是否有 EPA）
- 支持按市场过滤
