### 2.2 用户行为路径分析

```
用户触达 → 了解阶段 → 评估阶段 → 决策阶段 → 成交/流失

触达渠道：
  Google 搜索（品牌词/品类词）  ← 最主要，转化率最高
  社媒（LinkedIn/Instagram/Pinterest）  ← 视觉种草
  展会名片扫描  ← 意向明确，直接询价
  口碑推荐  ← 信任度高，转化快

关键流失节点：
  ① 首页加载超过3秒 → 直接关闭
  ② 无法快速判断"这家公司能不能做我的产品" → 离开
  ③ 没找到联系方式或询价表单 → 离开
  ④ 对质量不放心且无客户案例 → 离开
  ⑤ 报价等待时间过长 → 转向竞争对手
```

## 三、核心痛点与机会

### 3.1 目标用户的核心痛点

| 痛点 | 具体描述 | 网站如何解决 |
|---|---|---|
| **不了解定制流程** | "我有个想法，怎么变成实物？要走什么流程？" | 制作清晰的定制流程时间轴；每个步骤图文说明 |
| **担心质量** | "没有摸过实物，怎么相信质量？" | 高品质产品细节特写图；真实客户案例和见证；视频展示工厂实况 |
| **最小起订量门槛** | "我只做50枚，有工厂愿意接吗？" | 突出"支持小批量定制"（哪怕1枚起）；在首页和产品页明确标注MOQ |
| **设计不确定** | "我不知道怎么设计，只有概念想法" | 展示"设计辅助服务"；提供免费3D效果图预览的说明 |
| **报价不透明** | "发完询价要等好几天才有回复" | 提供在线即时询价表单；注明"24小时内回复"的承诺 |
| **不知道选什么工艺** | "金属、珐琅、电镀……我该怎么选？" | 制作"工艺对比表"页面；引导性问卷（"帮我选工艺"） |
| **不知道供应商靠不靠谱** | "找了个中国供应商，交了钱货不对版怎么办？" | 展示工厂资质认证、出口经验年限、真实客户Logo |
| **跨语言沟通障碍** | "我英文不好，能用其他语言沟通吗？" | 提供多语言版本（EN/ES/FR/AR可选）；WhatsApp/WeChat 等即时通讯入口 |

### 3.2 商业机会点

- 欧美市场专业奖章定制网站普遍**视觉粗糙**，大量小 B 客户（校友会、慈善机构、小企业）被大厂忽视，**小批量灵活定制**是差异化切入点
- 大多数中国制造商网站只做展示，没有**引导询价的路径设计**，询价转化率普遍低于 1%，优化后可提升至 3-5%
- **YouTube/Instagram 工厂视频**在 Pinterest 和 LinkedIn 的视觉内容分享能带来大量自然流量
- 与竞争对手相比，如果在 Google 搜索"custom metal medal manufacturer"进入前 3 名，每月可获得 **200-500 条高质量自然询价**

## 五、设计方向与风格定位

### 5.1 设计风格选择

本方案推荐采用**「高端工艺馆 × 现代商务」融合风格**，原因如下：

- 目标用户（欧美企业采购、政府官员）对设计质量有较高期望
- 金属徽章本身是"高品质、精密制造"的产品，网站视觉需呼应这种质感
- 既要体现工艺深度（工匠感），又要保持国际商务的专业可信

**不推荐**纯展示型风格（大量图片堆砌，无结构层次）和过于活泼的互联网风格（不符合 B2B 采购决策者审美）。

### 6.2 询价表单设计要点

**询价表单是全站最关键的转化组件，设计原则：**

1. **分步骤引导**：不一次性展示20个字段，让用户望而生畏
   - 步骤1（必填，3字段）：姓名 / 邮箱 / 产品类型（选择）
   - 步骤2（必填，3字段）：预计数量 / 预算范围 / 期望交货日期
   - 步骤3（选填，2字段）：产品描述 / 上传设计文件
   - 提交后：显示"我们将在24小时内回复您"的确认页

2. **表单字段设计**：
   - 数量字段：下拉选择（"50-100 / 100-500 / 500-1000 / 1000+"）而非自由输入，降低填写门槛
   - 产品类型：卡片式选择（带图标，如：奖章/徽章/纪念币/其他）
   - 文件上传：支持 JPG/PNG/PDF/AI，一步到位

3. **防流失设计**：
   - 每个页面底部固定「快速询价」浮动按钮（移动端尤为重要）
   - 表单填写超过 50% 时，弹出挽留提示（"还有问题？直接加微信：[微信号]"）

4. **提交后自动流程**：
   - 邮件通知客户（自动回复，告知将在24小时内回复）
   - 邮件通知管理员（新询价，包含完整信息）
   - CRM 记录（可选，如与微信机器人联动）

#### 定制流程页（Customization Process）

**设计目标**：消除首次客户对定制过程的陌生感和恐惧感。

```
[时间轴流程图 — 6步]

步骤 1：提交需求（1天内）
  └── 填写询价表单或发送邮件/WhatsApp
  └── 提供：产品类型、数量预算、设计想法或参考图

步骤 2：设计方案（1-3天）
  └── 设计师根据您的需求出初稿
  └── 免费提供3D效果图预览

步骤 3：确认打样（3-7天）
```

## 八、技术方案

### 8.1 推荐技术选型

| 组件 | 推荐方案 | 说明 |
|---|---|---|
| 网站框架 | **WordPress**（推荐）或 **Webflow** | WordPress 有丰富插件生态，后期客户可自行维护；Webflow 设计自由度更高 |
| 主题/模板 | 从头定制设计（非模板套用） | 确保差异化，避免与竞争对手雷同 |
| 页面构建 | Elementor Pro（WordPress）或 Webflow Native | 灵活拖拽，便于后期内容更新 |
| 托管 | **Cloudflare Pages**（免费）或 **Netlify** | 全球CDN，访问速度快 |
| CDN | Cloudflare（已含在托管方案）| 全球加速 + 免费SSL |
| 图片优化 | Cloudflare Images 或 ShortPixel | 压缩图片，加速加载 |
| 表单 | FluentForm（WordPress）或 Tally（独立表单） | 邮件通知 + 数据记录 |
| 在线客服 | Tawk.to（免费）或 WhatsApp浮窗链接 | 零成本接入 |
| 邮件营销 | Mailchimp 或 ConvertKit | 询价后续跟进自动化 |
| 数据分析 | Google Analytics 4 + Google Search Console | 流量监控 + SEO健康度 |
| SEO 插件 | Yoast SEO 或 Rank Math（WordPress） | 技术SEO配置 |

### 8.2 移动端适配要求

| 要求 | 说明 |
|---|---|
| 响应式断点 | 320px / 768px / 1024px / 1440px |
| 移动端优先 | 所有内容在手机上可读、可操作 |
| 图片策略 | 移动端使用压缩版本（WebP） |
| 触摸友好 | 按钮高度 ≥ 44px；间距 ≥ 8px |
| 快速加载 | 首屏内容 < 3秒（Lighthouse Mobile ≥ 80） |
| 表单体验 | 移动端键盘自动切换（文字/邮箱/数字） |

### 8.3 安全要求

- 全站 HTTPS（Let's Encrypt 或 Cloudflare 自动配置）
- WordPress 后台路径自定义（非默认 /wp-admin）
- 强密码 + 两步验证（管理员账号）
- 定期自动备份（UpdraftPlus，每日备份，保留7天）
- 表单添加 Google reCAPTCHA v3（防垃圾询价）
