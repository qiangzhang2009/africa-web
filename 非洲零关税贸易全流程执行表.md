# 二、最推荐 4 条赚钱路线（按成功率排序）

## 1. 咖啡生豆（埃塞/肯尼亚）

- 零关税：8% → 0%

- 适合：贸易、电商、烘焙店、私域

- 周期：15–30天起量

## 2. 可可/腰果/芝麻（科特迪瓦/加纳/坦桑尼亚）

- 零关税：8%–10% → 0%

- 适合：食品厂、批发商、直播带货

## 3. 鲜花/牛油果/南非柑橘（肯尼亚/南非）

- 零关税：8%–10% → 0%

- 适合：生鲜渠道、花店、社区团、电商

## 4. 矿产/橡胶/铝土矿（刚果金/几内亚/南非）

- 零关税：0%–5% → 0%

- 适合：大宗贸易、工厂原料采购

# 三、1 张表：从0到通关全流程（可直接打印）

## 1）前期准备（3–7天）

- [ ] 公司有：**进出口权 + 海关注册编码**

- [ ] 外币账户 / 可做跨境结算

- [ ] 食品/生鲜：查**中国海关准入名单**

- [ ] 确定 1–2 个主打品类

## 2）找货源 & 签合同（关键）

- [ ] 对接：农场/工厂/当地出口商

- [ ] 合同必须写：

- 卖方负责**原产地证书**

- 明确：**非洲原产**

- 付款方式：TT / 信用证

- 运输：CIF 上海/宁波/广州

## 3）发货前必办：原产地证（决定能不能零关税）

- [ ] 让出口商在当地办理：

- **官方原产地证书**

- 商业发票

- 装箱单

- 提单

- [ ] 证书上必须体现：**53个非洲建交国**

## 4）物流方案

- 大宗/原料：**海运**（18–30天）

- 生鲜/鲜花/高价货：**空运**（3–7天）

- 优先：**直航船/直航班机**

## 5）国内清关（走绿色通道）

- [ ] 报关时选择：**非洲零关税优惠**

- [ ] 提交：

1. 原产地证

2. 发票

3. 箱单

4. 提单

5. 检疫证书（食品/生鲜/植物）

- [ ] 走：**非洲输华绿色通道**

## 6）到货销售

- [ ] 分销给批发商/工厂

- [ ] 进保税仓做电商

- [ ] 国内加工再销售（利润更高）
